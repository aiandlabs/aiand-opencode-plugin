1 Install
curl -fsSL https://opencode.aiand.com/install.sh | bash
2 Sign in
$opencode auth login
3 Start
$opencode

Sign in picks ai& and takes your sk-… key — get one at aiand.com. Prefer to do it by hand? Add "@aiand/opencode-plugin" to your opencode.json plugin list.

Live catalog. New ai& models show up on their own. One key. Log in once, use it everywhere. OpenCode-native. Just a plugin — nothing to run.
