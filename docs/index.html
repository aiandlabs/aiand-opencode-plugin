ai& · OpenCode

Add ai& to OpenCode in one line

Installs OpenCode (if you don't have it) and wires up the ai& provider plugin — live model catalog and API-key auth through the ai& gateway.

$ curl -fsSL https://opencode.aiand.com/install.sh | bash

Then

  1. opencode auth login → choose ai& → paste your sk-… key
  2. opencode to start the TUI, or opencode run "say hi" for one-shot

Get a key at aiand.com. No key needed to browse the model catalog — only to send a message.

What the script does

  1. Installs OpenCode via its official installer if it isn't already on your PATH.
  2. Adds @aiand/opencode-plugin to your global OpenCode config (~/.config/opencode/opencode.json) — OpenCode fetches it from npm on next run.
  3. Sets a default ai& model, only if you don't already have one. Your config is backed up first, and re-running is safe.
Source & docs on GitHub. Prefer manual setup? Add "@aiand/opencode-plugin" to your opencode.json plugin list.
